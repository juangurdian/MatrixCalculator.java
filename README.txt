Instructions for Running the Matrix Calculator Program
Dear Professor,

Thank you for reviewing my project. This project is a Matrix Calculator implemented in Java. To execute the program, please follow these steps:
		Unzip the Project Archive:
	•	Locate the zip archive file that contains all the project classes.
	•	Extract the contents of the zip archive to a directory of your choice on your computer.
		Open Eclipse 
	•	Launch Eclipse once it's installed.
		Import the Project:
	•	In Eclipse, go to "File" > "Open Projects from File System..."
	•	Click on "Directory..." and select the directory where you extracted the project files.
	•	Click "Finish" to import the project.
		Run the Matrix Calculator:
	•	In the Eclipse Package Explorer on the left, locate the project folder (e.g., "MatrixCalculator").
	•	Expand the project to see the source files.
	•	Right-click on the "MatrixApp.java" file located under the "src" folder.
	•	Select "Run As" > "Java Application."
		Interact with the Program:
	•	The Matrix Calculator program's graphical user interface (GUI) should appear.
	•	Enter the matrix dimensions and choose an operation (Add, Subtract, Multiply).
	•	Click the "OK" button to enter matrix values and perform calculations.
	•	Alternatively, you can click the "Calculate" button to perform calculations with pre-defined matrix values.
	•	The "Clear" button clears the result area.
		Exit the Program:
	•	To exit the program, simply close the program's window.

Please let me know if you encounter any issues or have any questions about the program. Thank you for taking the time to review my project.
